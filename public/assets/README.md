# Assets Directory

Place all image and video assets in this directory structure.

## Required Assets

### Hero Images
- `/public/assets/hero-1.jpg` - Main hero image
- `/public/assets/hero-2.jpg` - Secondary hero image
- `/public/assets/hero-video.mp4` - Hero background video (optional)

### Product Cards
- `/public/assets/cards/card-1.jpg` - Product card image 1
- `/public/assets/cards/card-2.jpg` - Product card image 2
- `/public/assets/cards/card-3.jpg` - Product card image 3

### Branding
- `/public/assets/logo-stara.png` - Stara logo
- `/public/assets/og-stara.jpg` - OpenGraph image (1200x630px recommended)

### Documentation
- `/public/assets/specs/sample-spec.pdf` - Sample specification PDF

## Image Guidelines

- Use high-quality images (minimum 1920px width for hero images)
- Optimize images before adding (compress, use WebP where possible)
- Maintain aspect ratios:
  - Hero images: 16:9 or 21:9
  - Product cards: 4:3
  - OG image: 1.91:1 (1200x630px)

## Video Guidelines

- Use MP4 format for maximum compatibility
- Keep file sizes reasonable (< 10MB recommended)
- Use muted, loop, autoplay attributes for background videos
